One tool. Every API.
Your git repo.

Theridion is the open-source API testing platform that replaces Postman, SoapUI, and half your toolchain. REST, GraphQL, gRPC, SOAP, WebSocket, Kafka — all in a single desktop app, backed by plain files you own.

Get Started

Everything you need

🌐

Multi-protocol

REST, GraphQL, WebSocket, SOAP/WSDL, Kafka, gRPC — first-class support for every protocol, not just HTTP.

🤖

AI Test Generation

Point it at an endpoint and get a full test suite. Powered by Ollama — your prompts never leave your machine.

🔌

MCP Server

Expose your collections as an MCP tool. Claude Desktop can inspect, execute, and test your APIs directly.

🛡

Self-healing Assertions

When a response schema drifts, Theridion suggests updated assertions instead of just failing red.

🎬

Trace Viewer

Playwright-style HTML trace reports for every test run. Step through requests, inspect headers, replay failures.

📁

Git-native Collections

Collections are plain JSON files on disk. Branch, diff, merge, and review API tests like any other code.

🎨

10 Visual Themes

From pitch-black to high-contrast. Pick the palette that matches your terminal and your mood.

🖥

Network Console

Chrome DevTools-style timing breakdown: DNS, TCP, TLS, transfer. Know exactly where latency hides.

⌨

CLI Runner + HTML Reports

Run collections from CI with theridion test. Get JUnit XML or a rich HTML trace report.

Why switch?

Your data stays on your machine

No cloud sync, no telemetry, no vendor lock-in. Collections live as files in your project repository.

No cloud account required

Download, install, run. No sign-up wall, no team seat limits, no surprise invoices.

Git-friendly file format

Every request, environment, and test is a diffable JSON file. Code review your API tests in the same PR as your code.

Get started in 60 seconds

1

Clone

git clone https://github.com/ArtifactForms/theridion.git cd theridion/apps/desktop
2

Install

pnpm install pnpm sidecar:bundle
3

Run

pnpm tauri dev
